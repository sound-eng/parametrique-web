Install Parametrique

Step-by-step guide for macOS.

Before you start

Parametrique requires macOS 14.6 or later. The installer places the HAL audio driver and the Parametrique app on your Mac.

1. Download

Get the latest .pkg from the download page or GitHub Releases.

2. Run the installer

  1. Double-click Parametrique-<version>.pkg.
  2. Follow the installer prompts and enter your admin password when asked.
  3. When finished, the driver and app are installed system-wide.

3. Select the audio device

  1. Open System Settings → Sound.
  2. Under Output, choose Parametrique EQ Device.
  3. If output is silent, raise the Parametrique EQ Device level in Audio MIDI Setup.

4. Launch Parametrique and select target device

  1. Launch Parametrique from /Applications.
  2. Open Parametrique → Settings (menu bar) and under Proxied device, choose the physical output you want to hear — your speakers, headphones, or audio interface.
  3. Without this step, audio routed through Parametrique will go to first device in the list.

5. Configure EQ

  1. Adjust the 8-band EQ in the main window.
  2. Change buffer size or other driver options in Settings if needed.
  3. EQ remains active after you quit the app — reopen only to change settings.

Uninstall

Open Parametrique → Settings → Driver → Uninstall… and follow the embedded uninstaller. Afterwards, remove the app from /Applications.

Troubleshooting

  • No sound — confirm Proxied device is set in Parametrique → Settings, and raise Parametrique EQ Device level in Audio MIDI Setup.
  • Crackling or pops — increase buffer size in Parametrique → Settings.
  • Driver not found — reinstall from the latest .pkg release.
  • Diagnostics — Parametrique → Settings → Export Diagnostics.
