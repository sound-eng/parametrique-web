Parametrique

Virtual audio driver with per-channel parametric EQ for macOS. Menu bar volume works with interfaces that have no software volume control.

8-band parametric EQ Per-channel EQ with a live response graph. Settings persist after you quit the app.
System volume control Proxies audio so keyboard and menu bar volume work with your output device.
Driver + app One installer packages the HAL driver and the Parametrique configuration app.

Quick install

  1. Download and open Parametrique-<version>.pkg.
  2. Complete the installer (one admin password prompt).
  3. Open System Settings → Sound and select Parametrique EQ Device as output.
  4. Launch Parametrique from /Applications.
